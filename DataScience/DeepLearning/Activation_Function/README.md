## softmax

1. 수식
   - $$y_k = \frac{\exp(a_k)}{\sum_{i=1}^n\exp(a_i)}$$
1. 정의
   - 입력받은 값을 0~1사이의 값으로 정규화후 출력해 주고, 값의 합은 항상 `1`이된다
